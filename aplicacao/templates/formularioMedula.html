Perguntas:

1. Você teve febre nos últimos 7 dias?

Observação: Se você teve febre nos últimos 7 dias, aguarde até a completa recuperação para realizar a doação de sangue.

2. Nos últimos 12 meses, você viajou para algum país com alta incidência de malária?

Observação: Se você viajou para algum país com alta incidência de malária nos últimos 12 meses, aguarde 1 ano após o retorno para realizar a doação de sangue.

3. Você já teve hepatite após os 11 anos de idade?

Observação: Se você teve hepatite após os 11 anos de idade, não é elegível para realizar a doação de sangue.

4. Você já teve diagnóstico de sífilis?

Observação: Se você teve diagnóstico de sífilis, não é elegível para realizar a doação de sangue.

5. Você já teve diagnóstico de HIV (AIDS)?

Observação: Se você teve diagnóstico de HIV (AIDS), não é elegível para realizar a doação de sangue.

6. Você já fez uso de drogas ilícitas injetáveis?

Observação: Se você fez uso de drogas ilícitas injetáveis, não é elegível para realizar a doação de sangue.

7. Você já teve diagnóstico de doença de Chagas?

Observação: Se você teve diagnóstico de doença de Chagas, não é elegível para realizar a doação de sangue.
